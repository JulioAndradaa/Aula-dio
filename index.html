O que são tags HTML?

Uma tag Html faz parte da estrutuda que demarca os textos, e conteudos de uma pagina, com essas tags é possivel realizar comandos de acordo com os seus gostos. Ela é usada para colocar conteudos na pagina, sejam eles imagens, textos, botões, pragrafos, e é usada não só na linguagem de marcação de hipertexto, mas também em varias outras linguagens, porem cada linguagem tem a sua semântica, e suas tags expecificas.

Tag h1...h6

A tag h1 é uma tag usada para marcação de texto HTML que demarca titulo, ela contem outras variações que são h1, h2, h3, h4, h5, h6. Elas dasicamente faz a mesma coisa, a unica diferença entre elas é o tamanho da caixa de texto, h1 para a maior, e assim sussecivamente, até a h6, com essa variação possibilita que nós tenhamos um leque maior de escolha para demarcar titulos da pagina.

Tag p

A tag p é uma tag HTML ultilizada para colocar um paragrafo de texto na pagina web.

Tag mark

a tag mark deixa uma palavra, ou uma certa parte do texto, em destaque, marcado como referência.

Tag small

A tag small Possibilita que você reduza a fonte de uma palavra ou uma parte do um texto expecifico.

Tag i

A tag i possibilita que coloquemos uma palavra, ou uma certa parte do texto em ITALICO

Tag u

A tag u faz com que possamos subilinhar um elemento ou até mesmo uma palavra, ou uma certa parte do texto.

Tag strong

Quando usamos a tag strong, damos um significado a uma parte de um texto, dizemos que ela deve ser lida com uma certa importância, e ela aparece com mais intencidade.

Tag ol

Quando queremos fazer uma lista ordenada (lista ordenada significa que os itens são classificados em ordem) ex: 1º item, 2ºitem, 3ºItem... usamos a tag ol, essa tag é uma tag hierárquica, isso significa que ela é uma tag "Pai" e á tags que a subdivide em itens, as tags filhas são li, ela demarca os itens contidos na lista ordenada

Tag ul

Quando queremos fazer uma lista que não precisa ser ordenada, usamos a tag 'ul', essa tag assim como a 'ol' ela é uma tag de caracteristica hierárquica, isso faz com que ela também tenha tags filhas que são representadas pelas tags 'i'

Tag li

Tag li ultilizada para demarcar um item de uma lista, seja ela ordenada ou não ordenada.

Tag a

A tag 'a', é uma tag usada para criar uma âncora de onde vincular o conteudo, nessa tag é usado uma tributo chamado "Href" para poder endereçar o documento.

Tag hr

A tag hr introduz uma linha horizontal entre os conteudos, ou textos de uma pagina web.

" Tag sub

A funcionalidade dessa tag é deixar o texto alinhado e com um tamanho de fonte, menor que o do texto original.

Tag sup

A tag 'sup' faz o conteudo ficar em alinhamento vertical comparado ao texto principal.

"Tag Blockquote

Essa tag significa que um conteudo é uma citação.

Tag title "

A tag title é ultilizada dentro da tag 'head'(cabeça) para marcar o titulo da pagina, o nome que fica na aba do navegador. como o nome já diz: Titulo
